# 🌿 AgriMitra AI

An AI-powered Smart Farming Assistant built to help farmers with real-time monitoring, intelligent crop management, and AI-based decision support.

## 🚀 Features

- 🤖 AI Farming Assistant (OpenAI Powered)
- 🌦 Real-Time Weather Forecast
- 💧 Live IoT Monitoring using ESP32
- 📊 Smart Analytics Dashboard
- 👤 Farmer Profile Management
- 🔐 Secure Login & Authentication
- ☁ Firebase Realtime Database Integration
- 🖼 Cloudinary Image Upload
- 🧠 Machine Learning Crop Prediction

## 🛠 Technologies Used

### Frontend

### Artificial Intelligence
- OpenAI GPT

### Hardware
